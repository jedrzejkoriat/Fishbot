
# 🎣 FishBot

**Fishbot** is a bot for Metin2 game private server, designed to automatically catch fish in the game. It uses tesseract to read the notifications and reacts on those messages that notify about fish being on a hook.

## 📖 Instruction

### **1. Launch Metin2 client in a 1920x1080 window mode**
### **2. Login to the game**
### **3. Equip a fishing rod**
![image](https://github.com/user-attachments/assets/c515eb49-bace-493c-b1f5-fd05329e666f)
### **4. Place the bait under F3 key**
![image](https://github.com/user-attachments/assets/e44e6b5d-a88c-41c6-a438-dc78b3126d7f)
### **5. Open "Pokaż stare wiadomości" window**
![image](https://github.com/user-attachments/assets/d5892a58-31d1-4b93-bead-07182aa84f0e)
### **6. Select "Łowienie"**
![image](https://github.com/user-attachments/assets/69afeae0-0ec2-4ab1-8233-e743994fcb5e)
### **7. Move "Protokół wiadomości" window to the left bottom corner**
![image](https://github.com/user-attachments/assets/7c595ce4-54c2-4333-8fcd-6008392d174a)
### **8. Run "Fishbot.exe"**
### **9. Press "Select Window"**
### **10. Select your Metin2 client window**
![image](https://github.com/user-attachments/assets/5e40d829-63bd-4e67-8994-8a946cb051ab)
### **11. Press "Start"**
### First, it will place the bait on your rod (F3) and cast it into the water (Space), then wait for the text "Kliknij spację jeszcze X razy" in the first line from the bottom, read the numerical value (X), and press space X times. Then the cycle repeats.
### Enjoy! 🐟
